Basic overview of the model's features

1) word_count - Can be directly Calculated 
2) char_count - Can be directly Calculated 
3) has_media - Can be directly Calculated 
4) sentiment - import text blob then calculate
5) company_avg_likes - maintain a dictionary for the mappings then map it (if not there take avg value over all)
6) hashtag - Can be directly Calculated 
7) avg_likes_keyword - first find the keyword with tf idf then map it with the dictionary else take avg
